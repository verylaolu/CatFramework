# CatFramework
I AM CHINAESE<br>
这是闲来无事自己开发的一个PHP框架<br>
代码可能差一些， 但是还是兼容了很多的思想<br>
可扩展性，根据自己情况添加扩展<br>
功能包依赖注入，避免不必要的加载，浪费资源<br>
灵活，基类分为4个层级， 可根据复杂性灵活运用<br>
集成部分功能，模板引擎，MYSQL支持，REDIS支持，分布式的部分处理和很多功能包<br>
